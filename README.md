# WhatsApp Bulk Sender

Chrome-расширение для массовой рассылки сообщений через **WhatsApp Web** с шаблонами, очередями номеров и рандомизацией для минимизации риска блокировки.

## ✨ Возможности

- 📋 **Массовая рассылка** — списку номеров с настраиваемой задержкой
- 🎲 **Несколько шаблонов** — выбираются рандомно для каждого получателя (антиспам)
- 👥 **Очереди номеров** — сохранённые списки: «Клиенты 2024», «VIP», «Тест»
- 👤 **Подстановка имени** — `{name}` в тексте заменяется на имя из CSV-формата
- 📎 **Вложения** — отправка фото и видео
- 📊 **Прогресс** — счётчики, журнал, сохранение между сессиями
- 🗄️ **Архивация** — диалоги автоматически архивируются после отправки
- 🔄 **Авто-обновление** — проверка новых версий раз в 6 часов
- 🎨 **UI в казахском стиле** — teal/gold палитра, орнаментальные разделители

## 🚀 Быстрая установка (для разработки)

1. Скачайте репозиторий:
   ```bash
   git clone https://github.com/Mob2222/whatsapp-sender.git
   ```
2. Откройте `chrome://extensions`
3. Включите **«Режим разработчика»** в правом верхнем углу
4. Нажмите **«Загрузить распакованное»** → выберите папку `whatsapp-sender/`
5. Откройте [web.whatsapp.com](https://web.whatsapp.com), авторизуйтесь
6. Нажмите на иконку расширения в панели Chrome

## 📖 Использование

1. В поле **«Номера получателей»** вставьте номера (один на строку):
   ```
   77011234567
   87029876543,Иван
   77776543210,Айгерим
   ```
2. Создайте **шаблон сообщения** (кнопка «+ Добавить шаблон»). Используйте `{name}` для подстановки имени.
3. Отметьте галочкой один или несколько шаблонов — они будут чередоваться случайно.
4. (Опционально) Сохраните **очередь номеров** для повторного использования.
5. (Опционально) Прикрепите фото/видео.
6. Укажите задержку (рекомендуется **10+ секунд** между сообщениями).
7. Нажмите **«▶ Начать рассылку»**.

### Форматирование текста WhatsApp

- `*жирный*` → **жирный**
- `_курсив_` → *курсив*
- `~зачёркнутый~` → ~~зачёркнутый~~
- ` ``` код``` ` → `код`

## 🏢 Корпоративная установка (авто-обновление)

Для распространения на множество рабочих станций Windows без магазина:

1. Подпишите `.crx` ключом `.pem` (см. `publish.sh`)
2. Разместите `.crx` + `updates.xml` на публичном URL (GitHub Releases)
3. Запустите `install.bat` с правами администратора на каждой машине
4. Chrome сам установит и будет обновлять расширение через реестр

Подробнее — см. [`CHROME_STORE_GUIDE.md`](CHROME_STORE_GUIDE.md) раздел 11.

## 🛠️ Техническая архитектура

```
┌──────────────────────────────────────┐
│  popup.html / popup.js               │  ← UI в панели расширения
│  (chrome.storage.local)              │
└──────────────┬───────────────────────┘
               │ chrome.tabs.sendMessage
               ▼
┌──────────────────────────────────────┐
│  content.js                          │  ← мост в WhatsApp Web
│  (isolated world)                    │
└──────────────┬───────────────────────┘
               │ window.dispatchEvent
               ▼
┌──────────────────────────────────────┐
│  inject.js + wa-js@3.23.3            │  ← взаимодействие с API
│  (MAIN world, CSP снят через DNR)    │
└──────────────┬───────────────────────┘
               │ WPP.chat.sendTextMessage
               ▼
       WhatsApp Web internal API
```

**Технологии:**
- Manifest V3 + Service Worker
- [`@wppconnect/wa-js`](https://github.com/wppconnect-team/wa-js) — open-source библиотека WhatsApp Web API
- `declarativeNetRequest` для снятия CSP на `web.whatsapp.com`
- `chrome.storage.local` для шаблонов и очередей
- `chrome.alarms` для периодической проверки обновлений

## 📁 Структура репозитория

```
whatsapp-sender/            ← код расширения
├── manifest.json
├── background.js           ← service worker (авто-обновление)
├── content.js              ← мост popup ↔ inject
├── inject.js               ← WPP API-слой
├── popup.html / popup.js   ← UI
├── rules.json              ← правила DNR (CSP)
└── icons/                  ← 16/48/128 PNG

version.json                ← манифест обновлений для background.js
updates.xml                 ← манифест gupdate для ExtensionInstallForcelist
publish.sh                  ← скрипт релиза (bash)
install.bat / uninstall.bat ← корпоративная установка Windows
privacy.html                ← политика конфиденциальности (для GitHub Pages)
PRIVACY.md                  ← то же в Markdown
CHROME_STORE_GUIDE.md       ← руководство публикации
```

Политика конфиденциальности

Расширение WhatsApp Bulk Sender • Последнее обновление: 14 апреля 2026 г.

1. Общие положения

Настоящая Политика конфиденциальности описывает, какую информацию собирает, использует и хранит расширение браузера Google Chrome «WhatsApp Bulk Sender» (далее — «Расширение»).

Используя Расширение, Вы соглашаетесь с условиями настоящей Политики.

2. Какие данные собираются

Расширение НЕ собирает и НЕ передаёт персональные данные на внешние серверы. Все данные хранятся локально на устройстве пользователя через Chrome Storage API (chrome.storage.local) и никогда не покидают устройство пользователя, за исключением случаев, описанных в разделе 5.

2.2. Что НЕ собирается:

  • Личные данные пользователя (имя, email, телефон аккаунта)
  • История переписки в WhatsApp
  • Учётные данные, пароли, cookies, токены
  • Геолокация
  • Данные о других сайтах, посещаемых пользователем
  • Аналитика, телеметрия, рекламные идентификаторы

3. Как используются данные

Введённые пользователем данные используются только для выполнения функций Расширения: отправка текстовых сообщений и медиа через открытую вкладку web.whatsapp.com, подстановка имени получателя в шаблон, отображение статистики, сохранение шаблонов и очередей.

Никакой обработки, агрегирования, анализа или продажи данных не производится.

4. Передача данных третьим лицам

Расширение не передаёт пользовательские данные третьим лицам, рекламным сетям, аналитическим системам или разработчику.

5. Внешние сетевые соединения

Расширение выполняет ограниченный набор внешних соединений, не содержащих пользовательских данных:

5.1. Загрузка библиотеки WhatsApp API (wa-js)

При первом открытии web.whatsapp.com Расширение загружает open-source библиотеку @wppconnect/wa-js с публичного CDN:

https://cdn.jsdelivr.net/npm/@wppconnect/wa-js@3.23.3/dist/wppconnect-wa.js

Библиотека необходима для взаимодействия с внутренним API WhatsApp Web. При этой загрузке передаются только стандартные HTTP-заголовки браузера (User-Agent, IP). Номера, тексты и вложения не отправляются.

Исходный код библиотеки: github.com/wppconnect-team/wa-js

5.2. Проверка обновлений

Раз в 6 часов Расширение запрашивает файл version.json с настроенного сервера обновлений, чтобы узнать актуальную версию. Пользовательские данные не передаются.

5.3. Отправка сообщений через WhatsApp Web

Сами сообщения отправляются через официальный веб-интерфейс WhatsApp Web в контексте Вашего аккаунта WhatsApp — так же, как если бы Вы отправляли их вручную. Разработчик Расширения не имеет доступа к Вашему аккаунту, номеру, переписке или содержимому сообщений.

7. Безопасность

  • Все пользовательские данные хранятся в зашифрованном профиле Chrome.
  • Расширение не использует внешние серверы для хранения данных.
  • Открытый исходный код расширения доступен для аудита.

8. Права пользователя

Вы можете в любой момент:

  • Удалить все сохранённые данные, удалив расширение через chrome://extensions.
  • Очистить отдельные шаблоны или очереди внутри интерфейса (кнопка ✕).
  • Отключить авто-проверку обновлений, отключив расширение.

9. Дети

Расширение не предназначено для использования лицами младше 18 лет и не собирает данные от детей.

11. Изменения в Политике

При внесении изменений обновлённая версия публикуется по тому же URL, дата в шапке обновляется. Существенные изменения объявляются в релизных заметках.
